# Bordas

As propriedades de arredondamento e espessura das bordas dos elementos estão diretamente
relacionadas à sua definição. Essas características ajudam a diferenciar os elementos na
composição da interface visual, contribuindo para uma estética mais harmoniosa e intuitiva.

## Arredondamento

Esse conceito está relacionado à propriedade `border-radius` do CSS, que permite aplicar um
estilo visual de arredondamento nos cantos dos elementos. Dependendo do token selecionado,
podemos ajustar o grau dessa curvatura, tornando o elemento mais ou menos arredondado.

As definições disponíveis são:

## Espessura

Esse conceito está relacionado à propriedade `border-width` do CSS, que permite aplicar uma
espessura visual nas bordas dos elementos. Dependendo do token selecionado, podemos ajustar
a espessura da borda, tornando o elemento mais ou menos destacado.
